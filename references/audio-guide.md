# Audio Guide

Use with `[skill:seedance-audio]`.

Audio layers: dialogue, ambience, SFX, music/rhythm, and reference audio. Assign each layer explicitly.

Lip-sync pattern: `[Character A] says: "short line." Camera motion is minimal; mouth movement follows the dialogue; body action pauses slightly on key syllables.`

Beat-sync pattern: `[Audio1] defines rhythm. Object motion hits strong beats; camera move changes only on bar transitions; no extra character action competes with the beat.`
