# Image-to-Video Guide

## Core Rule

Prompt only what the image cannot show.

Do not re-describe visible static details unless they must be locked. Add motion, camera, timing, transformation, lighting change, and audio.

## Minimal Template

`[Image1] preserved. The subject remains visually consistent. [One motion]. [One camera move]. [Lighting change]. [Audio cue]. [Constraint].`

## Failure Fixes

- If identity drifts: reduce new visual description.
- If camera jumps: use one camera move.
- If product warps: say preserved, static identity, no shape change.
- If output is still: add one physical action and one time cue.
