# seedance-recipes

Use this skill for reusable Seedance 2.0 prompt templates. Recipes must be IP-safe, reference-aware, and short enough to adapt.

Return: genre/use case, asset roles, compact template, optional Chinese-compressed variant, safety notes, retry variant.

Safe recipe families: product ad, lifestyle, drama, music/beat sync, landscape, commercial/social, original animation, VFX.

Animation recipe rule: use original characters and production descriptors only: line weight, cel shading, smear frames, palette, texture. Do not name studios or franchises.

Legacy details moved to `references/migrated/seedance-recipes-original.md`; check them for IP labels before reusing.
