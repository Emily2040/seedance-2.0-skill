# seedance-recipes

Use recipes as starting patterns, not rigid prompt templates.

## Recipe Families

- Product: object fidelity, material, light pass, hero move.
- Lifestyle: action, environment, handheld realism.
- Drama: emotional beat, blocking, motivated camera.
- Music video: rhythm, gesture, cut energy.
- Landscape: atmosphere, slow camera, natural sound.
- Commercial: brand-safe product function, controlled polish.
- Animation: original character archetype, medium, motion style.

## Output Contract

Return one recipe, the prompt skeleton, and a compact final prompt.
