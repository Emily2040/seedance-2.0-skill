# seedance-motion

Use physical verbs and consequences. Motion should be observable on screen.

## Motion Contract

State: actor/object, action, force level, timing, physical consequence, and continuity requirement.

For reference footage, use only owned, licensed, public-domain, stock, mocap, rehearsal, or self-recorded material. Map `[Video1]` to motion, not identity, unless the identity is authorized.

## Output Contract

Return the motion phrase, reference role map if any, and repaired prompt language.
